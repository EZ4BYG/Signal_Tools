# Fourier-Transform
主要为"离散傅里叶变换(DFT)与卷积(conv)"相关matlab和python程序

说明1：DFT程序中读取外部txt数据文件，格式只有一个要求：**每行数据个数一致**。       

说明2：二维卷积只有一个要求：二维数据矩阵尺寸 **>** 卷积核尺寸(边长最好为奇数)。
